fn make_dev_inode( &self, _name: &str, _marker: usize ) -> Result<CacheArc<CacheItem<(usize, usize), CachedINode>>, FileSystemError>

Creates a new dev inode with the provided name and the device marker in the filesystem.

Overview

In the inner implementation this simply looks up for the device with the device marker in the global devices b-tree map and adds it as a device inode in the children array of itself.
